WiSAR3D - Aerial LiDAR dataset for 3D object detection

Anonymous
Paper arXiv Code Data

The WiSAR3D dataset includes LiDAR point clouds captured by an aerial drone, specifically designed for Wilderness Search And Rescue (WiSAR) scenarios. It features annotations for 3D object detection across a range of categories and sizes.

Abstract

Wilderness Search and Rescue (WiSAR) operations face significant challenges in locating and rescuing individuals in remote, rugged environments. Existing datasets focused on roadways or urban scenarios do not adequately address the complexities of wilderness settings, such as dense foliage and varied topography. To fill this gap, we introduce \textit{WiSAR3D}, the first aerial LiDAR 3D dataset tailored for WiSAR, collected in natural wilderness environments. \textit{WiSAR3D} consists of contiguous point clouds collected along flight paths, with multiple returns enabling to see under foliage. Additionally, we provide an analysis of the dataset and the first benchmark for aerial 3D detection. The results suggest that special models are needed in this domain, as finding individuals is still challenging.
